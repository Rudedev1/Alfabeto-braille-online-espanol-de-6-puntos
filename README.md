# Alfabeto-braille-online-español-de-6-puntos
Alfabeto braille online español de 6 puntos elaborado con las fuentes tipográficas gratuitas de la comisión braille española (CBE) que ofrecen en (https://www.once.es/servicios-sociales/braille/comision-braille-espanola), y siguiendo las directrices del documento técnico B15 de escritura con la fuente braille de la CBE (https://www.once.es/servicios-sociales/braille/comision-braille-espanola/documentos-tecnicos/documentos-tecnicos-relacionados-con-el-braille/documentos/b15-escritura-con-fuente-braille-v2.pdf)

La página tiene una función para cambiar las distintas fuentes de braille que la CBE ofrece, la normal, la enmarcada, la guiada y la enmarcada-guiada.

## Notas sobre la fuente digital
Note que los dígitos y las mayúsculas están escritas para representar los elementos aislados, **ésta no es la forma correcta de escribirlos en braille**. Para marcar digitos numéricos se usa el prefijo de número (Que en la fuente corresponde a #), seguido de los números, sin volver a repetir el prefijo contínuamente, mismo para las letras mayúsculas, que usan el prefijo que se marca con {.

# Ejemplo:

## Correcto:

- Palabra: salida
- Escritura: { s a l i d a
 
## Incorrecto:
- Escritura: { s { a { l { i { d { a

# Antes de usar, tenga en cuenta:

- Los carácteres de la fuente no representan exactamente los del braille de verdad, para ésto debe consultar el documento técnico antes mencionado. Entre lo destacable está:
- Los números no se escriben del 1 al 9, sino de la a a la f (Punto 5 - Escritura de los números)
- Si no dispone de letra de c trencada (Ç), puede usar & para representarla en la fuente.

Consulte las 64 combinaciones de 6 puntos y el documento completo para saber a que caracteres equivale en la fuente y otras pautas para escribirla correctamente.
